Place the gallery images from the Prestige Kitchens email in this folder.

Rename them with these prefixes before wiring the gallery UI:

- `ki-` for kitchens
- `ba-` for bathrooms
- `be-` for bedrooms

Current decision:

- build a single gallery page first
- do not add gallery filters yet
- most images are kitchens, with one bathroom image mixed into the same gallery for now

Example filenames:

- `ki-01.jpeg`
- `ba-01.jpeg`
- `be-01.jpeg`

The Gmail connector in this session can list the image attachments but cannot export `.jpeg` or `.png` binaries, so the files still need to be copied in manually.
